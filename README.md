# Movie Search App

## 📌 Description

Movie Search App is a React application for searching movies by keyword.  
Users can enter a movie title or keyword and get a list of matching movies from an external API.

The application supports pagination, displays notifications for user interactions, and provides a simple and convenient interface for browsing search results.

## 🚀 Live Demo

[View Demo](https://03-react-movies-andrii888.vercel.app/)

## 🛠 Technologies

- React
- TypeScript
- Axios
- React Query
- React Paginate
- React Hot Toast
- Vite
- CSS

## ✨ Features

- Search movies by keyword
- Fetch movie data from an external API
- Display a list of matching movies
- Pagination for browsing search results
- Loading and error handling
- Toast notifications for user interactions
- Responsive and user-friendly interface
